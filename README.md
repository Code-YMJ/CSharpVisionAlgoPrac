# CSharpVisionAlgoPrac
## A repository for practicing computer vision algorithms using C#.
This repository contains implementations of various computer vision algorithms using the C# programming language. It serves as a learning resource for developers interested in computer vision and image processing.
### Contents
1. 각 부품의 위치 인식해서 사각형으로 ROI 표시
2. 각 부품의 정상품(파랑색 ROI)과 불량품 위치(빨강색 ROI) 인식 
3. 각 부품의 회전 각도 측정 및 사이즈 측정
